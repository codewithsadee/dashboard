### Run Locally

To run **Dashboard** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/dashboard.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/dashboard.git
```
